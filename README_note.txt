小流量的数据处理框架。

概要： 每个算法将对一批数据进行数据处理，链式处理。

关键词：
        数据来源

        {
        数据模型
        数据算法
        }

        数据目标

1、数据的输入与输出
    数据模型 <-- 转换 --> 源
    数据算法 <-- 转换 --> 算法

    数据模型： 接受上流的数据（数据来源 或 数据算法），并进行数据规范化。在发送到数据算法上。
              格式为一个List<Data>,即一个批次。

    数据算法： 接受数据模型的输入，对应的输出将会直接输出到端口 或 数据模型中。
               在设计的时候，使用Sleep代表调用具体算法占用的资源（CPU、时间）。
